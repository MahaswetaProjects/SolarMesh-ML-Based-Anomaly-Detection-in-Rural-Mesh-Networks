# SolarMesh-ML-Based-Anomaly-Detection-in-Rural-Mesh-Networks

## Problem Statement  
Solar-powered rural mesh networks are critical for providing connectivity in underserved regions. However, they face frequent challenges:  
- **Unpredictable faults** due to environmental conditions and hardware failures.  
- **Lack of proactive monitoring**, leading to delayed fault detection.  
- **Reduced reliability and uptime**, which impacts education, healthcare, and digital inclusion in rural areas.  

## Contributions 
- Designed and implemented an **ML-based anomaly detection pipeline** to identify faults in solar-powered mesh networks.  
- Conducted **data preprocessing and feature engineering** on telemetry datasets to improve model accuracy.  
- Built **supervised and unsupervised ML models** for anomaly detection, ensuring scalability and deployment readiness.  
- Developed **visualization dashboards** (Power BI, Python plots) to interpret anomaly trends and network health.  
- Documented workflows for reproducibility and potential real-world deployment.  

## Tech Stack  
- **Programming:** Python (Pandas, NumPy, Scikit-learn, TensorFlow, Matplotlib, Seaborn)  
- **Data Analytics:** Power BI, Exploratory Data Analysis (EDA), Feature Engineering   
- **Tools:** GitHub, Google Colab 

## Results Achieved  
- Achieved **high anomaly detection accuracy** across solar mesh telemetry datasets.  
- Enabled **early fault identification**, reducing downtime and improving network reliability.  
- Demonstrated potential for **scalable deployment** in rural connectivity projects.  
- Provided **interpretable dashboards** for actionable insights into network health.
